# Campus Event Management Ticketing System

A comprehensive campus event management platform designed to simplify the organization and participation of university events.

The system provides a centralized solution for managing events, ticket reservations, attendee registration, and event monitoring through an intuitive and efficient interface.

Students can explore upcoming events, view event details, reserve tickets, and track their registrations, while administrators can create events, manage capacities, monitor attendance, and oversee event operations from a single platform.

## Key Capabilities

- Event creation and management
- Ticket reservation and registration
- Attendee tracking and monitoring
- Event capacity management
- Booking and reservation management
- User and participant administration
- Real-time event information access
- Centralized event organization workflow

## Purpose

The project aims to improve the efficiency of campus event organization by providing a digital solution that streamlines event planning, ticket distribution, participant management, and overall event coordination.

## Technology Stack

- Java
- MySQL
- JDBC
- IntelliJ IDEA
